# local-turistico

Landing Page de turismo desenvolvida como desafio prático do módulo de Fundamentos de HTML e CSS (Nível 3) do curso Full Stack da Rocketseat.

## Sobre o projeto
Este projeto consiste na criação de uma página web com informações sobre um local turístico, destacando pontos de interesse por meio de imagens atrativas e descrições informativas. A proposta é simular uma landing page voltada para apresentação de destinos, com foco em organização visual e experiência do usuário.

## Objetivos
- Praticar a estruturação semântica com HTML;
- Aplicar estilização utilizando CSS;
- Desenvolver um layout limpo e organizado;
- Trabalhar com imagens e conteúdos descritivos;
- Reforçar conceitos fundamentais de desenvolvimento web.

## Tecnologias utilizadas
- HTML5
- CSS3

## Aprendizados
Durante o desenvolvimento deste projeto, foram explorados conceitos essenciais como:
- Uso de tags semânticas;
- Organização de conteúdo em seções;
- Estilização com CSS;
- Boas práticas de estruturação de páginas;
- Criação de layouts visualmente agradáveis.

## Resultado
A página apresenta um layout focado na divulgação de um local turístico, com seções bem definidas, imagens ilustrativas e textos que enriquecem a experiência do usuário.
